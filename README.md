# StoneMetal Market

Локальный прототип интернет-магазина материалов: гранит, плитка, слэбы, профлист и трубы.

## Как открыть

Открой файл:

```text
C:\Users\Ivan\Documents\Codex\2026-04-26\new-chat\granite-industrial-store\index.html
```

Сайт работает без сервера и установки зависимостей.

## Демо-админ

```text
Email: admin@stonemetal.ru
Пароль: admin123
```

После входа появится раздел **Админка**.

## Что уже есть

- каталог товаров;
- фильтр по группам;
- поиск;
- сортировка;
- корзина;
- регистрация покупателей;
- вход администратора;
- создание и редактирование групп;
- создание и редактирование карточек товаров;
- загрузка картинок на группы и товары;
- оформление заявки из корзины;
- список заявок в админке;
- темная/светлая тема;
- адаптивная верстка для телефона.

## Где хранятся данные

Все данные сохраняются в браузере через `localStorage`. Это удобно для прототипа: можно быстро проверять админку и каталог без базы данных.

Для настоящего интернет-магазина следующий шаг: добавить backend, базу данных, хранение изображений на сервере и нормальную авторизацию.
